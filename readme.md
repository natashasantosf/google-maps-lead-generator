📌 Google Maps Business Lead Scraper

🚀 Problem

Many small businesses and agencies need structured lead lists from Google Maps but collecting this data manually is slow and inefficient.

💡 Solution

This Python-based automation script extracts business information directly from Google Maps search results and exports it into a clean CSV file.

The scraper collects:
- Business Name
- Phone Number
- Website
- Address

🛠 Tech Stack
- Python
- Selenium
- Pandas
- CSV

⚙️ How It Works

1. User inputs a search keyword (e.g., “Dentists in Miami”)
2. The script opens Google Maps
3. Scrolls through results
4. Clicks each business
5. Extracts structured information
6. Exports everything to a CSV file

📊 Output Example
| Name | Phone | Website | Address |
